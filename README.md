# InAmigos NGO Awareness Webpage

A responsive NGO awareness webpage created using HTML and CSS to spread awareness about InAmigos Foundation and its social initiatives.

## 🌟 Features

- Responsive modern design
- NGO introduction section
- Ongoing projects showcase
- Social impact section
- Gallery section
- Volunteer call-to-action
- Social media links
- Smooth UI and hover effects

## 💻 Technologies Used

- HTML5
- CSS3

## 📌 Purpose

This project was created to spread awareness about the social initiatives and campaigns of InAmigos Foundation.
